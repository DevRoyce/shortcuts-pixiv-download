# 基于ios捷径的pixiv下载器

自动分析内容类型并下载
支持：单张、画集、动图（转Gif）

使用方法：
可以通过 分享 或者 复制链接手动触发

安装地址：https://www.icloud.com/shortcuts/07de573e629f490d9828d4c20611be87

## 关于获取cookie的方法

这里以Chrome浏览器为例

1. 首先你要先在电脑上登录pixiv
2. 右键页面选择 “检查” 
3.  “Application” -> “Cookies” -> “https://www.pixiv.net”
4. 将 “PHPSESSID” 项的内容复制到捷径中
